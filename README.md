# Sky Logitech Macro Converter

## 📖 專案簡介
本工具的唯一目的，是將 **JSON 格式的數字樂譜** 轉換成 **Logitech 巨集格式**，以便在 Logitech 裝置上自動演奏。  
它不包含任何額外功能，也不涉及遊戲修改，只是單純的格式轉換器。

## 🎯 功能
- 讀取 JSON 樂譜檔案（數字表示音符）。  
- 轉換為 Logitech 巨集可用的指令序列。  
- 輸出結果可直接匯入 Logitech G Hub 或相關軟體。  

## ⚠️ 注意事項
- 本專案僅提供 **格式轉換**，不包含樂譜編輯器或播放器。  
- 請自行準備合法的 JSON 樂譜檔案。  
- 本工具與遊戲開發商、Logitech 官方無任何關聯。  

## 🚀 使用方式
1. 準備一份 JSON 樂譜檔案。  
2. 將檔案放入專案目錄並執行轉換程式。  
3. 取得輸出的 Logitech 巨集檔案，匯入 G Hub 使用。
